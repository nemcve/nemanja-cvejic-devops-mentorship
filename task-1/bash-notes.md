```
$ ssh <username>@<ipaddress> -p <port> - koristi se za povezivanje sa udaljenim mašinama
```
```
$ ls - koristi se za izlistavanje sadržaja foldera
```
```
$ cat - koristi se za prikazivanje sadržaja fajla
```
```
$ file - koristi se za dobijanje informacija o obliku fajlova i kako su enkodirani
```
```
$ find - koristi se za rekurzivnu pretragu fajlova
```
```
$ grep - koristi se za pretragu teksta output-a komandi kao što su cat, ls, ps...
```
```
$ strings - koristi se za ekstraktovanje teksta iz binarnih fajlova 
```
```
$ base64 - koristi se za dekodiranje i enkodiranje teksta iz fajlova
```
